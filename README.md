## ✨ Key Features

- **Role-Based Architecture:** Distinct, secure routing and dashboards for Students, Tutors, and NGOs.
- **Dual Authentication:** Secure local login (Email/Password) and OAuth 2.0 integration via Google.
- **Smart Onboarding:** Adaptive profile completion workflows based on user roles.
- **JWT Session Management:** Stateless, secure authentication using `HttpOnly` cookies.
- **Modern UI:** Responsive, accessible, and fast interface built with Tailwind CSS v4.

### Prerequisites

Ensure you have the following installed on your local machine:
- [Node.js](https://nodejs.org/en/) (v16.x or higher)
- [Git](https://git-scm.com/)
- A [MongoDB](https://www.mongodb.com/) account and cluster URL
- Google Cloud Console credentials (for OAuth)
